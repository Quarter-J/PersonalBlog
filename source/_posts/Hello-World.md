## 欢迎 👋

你好！欢迎来到我的个人博客。

这是我使用 **Hexo** 框架和 **Butterfly** 主题搭建的个人博客，托管在 **GitHub Pages** 上。

## 关于这个博客

这个博客将用于记录：

- 📝 技术学习笔记
- 💡 项目开发经验
- 🌟 生活中的思考和感悟
- 📚 读书笔记

## 技术栈

- **框架**: [Hexo](https://hexo.io/zh-cn/) - 快速、简洁且高效的博客框架
- **主题**: [Butterfly](https://butterfly.js.org/) - 美观、功能丰富的 Hexo 主题
- **托管**: [GitHub Pages](https://pages.github.com/) - 免费的静态网站托管服务
- **评论**: 待添加
- **搜索**: 待添加

## 开始写作

```bash
# 创建新文章
hexo new "文章标题"

# 本地预览
hexo server

# 生成静态文件
hexo generate

# 部署到 GitHub Pages
hexo deploy
```
